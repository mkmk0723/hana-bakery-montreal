# HANA Bakery & Pâtisserie — Montréal Version

Static HTML/CSS/JS portfolio site.

## GitHub Pages setup

Use this structure directly at the repository root:

- `index.html`
- `about.html`
- `menu.html`
- `shop.html`
- `news.html`
- `contact.html`
- `recruit.html`
- `css/style.css`
- `js/script.js`
- `images/`

GitHub Pages:

- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

This version includes cache-busting query strings for CSS and JS: `?v=20260708`.
